# Notes CRUD App
This is a website I built for the self learning purposes. Here data can be inserted, fetched, updated and can be deleted. Firebase is used for storing data. Firebase authenthication with email and password is done.
1. Account Registration and login:
   Account is registered using email and password.

2. Addition of Notes:
    Notes with title and content can be inserted.
3. Display Of notes:
    All notes created by the user is displayed.
4. Deletion:
    Notes can be deleted by the user
5. Editing The Note:
    Notes title and content can be edited.

## Prerequistics
Before you begin, ensure you have the following installed on your system:

Node.js (version 14 or higher)
npm (version 6 or higher) or Yarn

### Install dependencies

Open your terminal and run the following command to install dependencies:

```
npm install

```

### Run the server:

This command will run the server in development mode.
```
npm run dev

```
